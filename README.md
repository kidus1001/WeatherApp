# **Weather App**

A simple, beginner-friendly weather application built with HTML, Bootstrap, and JavaScript. 
Users can enter a city name to fetch and display real-time weather data (like temperature, humidity, 
wind speed, and description) using the OpenWeatherApp API (https://openweathermap.org/api).


## **Features**
- Search for current weather by city name
- Real-time temperature (°C), humidity, wind speed, and conditions
- Clean and responsive UI with Bootstrap
- Error handling for invalid input or city not found

## **Getting Started**

### **Prerequisites**

- A modern web browser
- OpenWeatherApp API Key (https://home.openweathermap.org/users/sign_up)


### **Setup**

1. Clone the repo:
   ```bash
   git clone https://github.com/your-username/weather-app.git
   cd weather-app
   
2. Open the index.html file in your browser.

3. In the script/script.js file, replace the placeholder API key.
   ```bash
   const apiKey = "YOUR_API_KEY_HERE";
